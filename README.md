# 🚀 LevonX – AI-Powered React UI Library (SaaS)

LevonX is a **full-stack AI-powered UI library platform** where developers can install ready-made React components via NPM or generate custom UI components using AI 🤯

This project is a **real-world SaaS application** combining AI, authentication, and payments — designed for developers who want to build and monetize modern tools 💰

---

## 🔥 Features

* 📦 Publish your own React UI Library to NPM
* 🤖 Generate UI components using AI (OpenRouter API)
* ⚡ Full MERN Stack application
* 💳 Razorpay Payment Integration (Subscriptions / SaaS)
* 🔐 Authentication & User Management
* 🎨 Prompt-based dynamic UI component generation

---

## 🧠 Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **AI Integration:** OpenRouter API
* **Payments:** Razorpay
* **Package Distribution:** NPM

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/labhamsharma1633/LevonX.git
cd LevonX
```

---

### 2️⃣ Setup Backend

```
cd LevonX-server
npm install
```

Create a `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
RAZORPAY_KEY=your_key
OPENROUTER_API_KEY=your_api_key
```

Run server:

```
npm start
```

---

### 3️⃣ Setup UI Library

```
cd LevonX-lib
npm install
```

---

## 📦 Publishing to NPM (Optional)

```
npm login
npm publish
```
